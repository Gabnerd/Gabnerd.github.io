RocketStar

RocketStar é uma jogo demo para gameboy desenvolvimento por mim para testar uma mecanica de jogo que estava pensando por um tempo em recriar e tambem para testar as minhas habilidades de aprender algo novo em pouco tempo.

Ele foi desenvolvimento usando uma linguagem chamada assembly GBZ80, derivada do assembly Z80. Como os outros assembly, é necessario entender sobre como a maquina funciona para conseguir usar o basico da linguagem.

tecnologias: Assembly GBZ80

RPGExpressSystem

Tecnicamente isso é um webapp, ele não é estatico, sua ideia é usar a biblioteca socket para criar uma ficha de dados que mudam em tempo real conforme o mestre queira.

Para que tivesse uma forma de armazenar os dados e serem lidos depois entre os jogadores e o mestre foi usado um arquivo JSON que era lido e reescrito constantemente.

tecnologias: HTML, CSS, JS, NodeJS, Socket, JSON(como banco de dados)
